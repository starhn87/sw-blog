## 블로그를 새로 만들었어요

안녕하세요, 이승우라고 해요.

기존에 Velog에서 글을 쓰고 있었는데, 직접 블로그를 만들어보고 싶다는 생각이 들었어요.
그래서 이번에 제로베이스에서 새로운 블로그를 구축하게 되었어요.

### 어떤 기술을 사용했나요?

이 블로그는 아래 기술 스택으로 만들어져 있어요:

- **Next.js 15** — App Router 기반으로 구성했어요
- **MDX** — 마크다운에 React 컴포넌트를 섞어 쓸 수 있어서 좋아요
- **Tailwind CSS** — 유틸리티 기반 스타일링이라 빠르게 작업할 수 있었어요
- **Framer Motion** — 인터랙티브한 애니메이션을 넣고 싶었어요

### 코드 예시

간단한 예시 하나 보여드릴게요.

```tsx
interface Props {
  name: string;
  count?: number;
}

export function Greeting({ name, count = 0 }: Props) {
  return (
    <div className="flex flex-col gap-2">
      <h1>Hello, {name}!</h1>
      {count > 0 && <span>{count}번째 방문이에요</span>}
    </div>
  );
}
```

### 앞으로의 계획

생각보다 직접 블로그를 만드는 게 재밌어서, 앞으로 기능을 더 추가해볼 예정이에요.

- AI 챗봇 — 블로그 콘텐츠 기반으로 질문에 답해주는 챗봇을 붙이려고 해요
- 댓글 시스템 — 방문자분들과 소통할 수 있으면 좋겠어요
- 검색 기능 — 글이 많아지면 필요하겠죠?

하나씩 만들어가면서 과정도 블로그에 기록해볼게요. 감사합니다!
