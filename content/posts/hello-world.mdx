## 블로그를 시작합니다

안녕하세요! MDX 기반의 새로운 블로그입니다.

### 기술 스택

이 블로그는 다음 기술로 만들어졌습니다:

- **Next.js 15** — App Router 기반
- **MDX** — 마크다운 + React 컴포넌트
- **Tailwind CSS** — 유틸리티 기반 스타일링
- **Framer Motion** — 인터랙티브 애니메이션

### 코드 예시

```typescript
function greet(name: string): string {
  return `Hello, ${name}!`;
}

console.log(greet("World"));
```
